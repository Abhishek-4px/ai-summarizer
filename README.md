# AI Text Summarizer

A FastAPI-powered text summarizer using Google Gemini API with structured JSON output.

## Features
- Summarizes text into 2-3 concise sentences
- Extracts 3-5 key points
- Returns structured JSON with summary, key points, and word count
- Beautiful dark-themed frontend
- Copy JSON to clipboard

## Setup
Set `GEMINI_API_KEY` as a Secret in your Hugging Face Space settings.
